# 一键数据库部署指南

## 📋 概述

`online_exam_one_click.sql` 是当前项目唯一保留的一键导入数据库文件。

它已经整合了：
- 全量基础表结构
- 客观题题库 660 道
- 主观题题库 36 道
- 主观题答案、答题记录、通知、管理员通知历史等当前代码实际使用的表
- 学生、教师、考试、成绩和留言测试数据

## 🗃️ 数据库结构

### 核心表结构
- **admin** - 管理员信息表
- **teacher** - 教师信息表  
- **student** - 学生信息表（支持BIGINT学号）
- **exam_manage** - 考试管理表
- **score** - 成绩管理表（支持BIGINT学号）

### 题库表结构
- **multi_question** - 选择题题库（支持10字符难度等级）
- **fill_question** - 填空题题库（支持10字符难度等级）
- **judge_question** - 判断题题库（支持10字符难度等级）
- **paper_manage** - 试卷管理表

### 辅助表结构
- **answer_record** - 客观题答题记录表
- **essay_answer** - 主观题答案与评分表
- **message** - 留言表
- **replay** - 回复表
- **notification** - 用户通知表
- **admin_notification** - 管理员通知历史表

## 🚀 部署步骤

### 1. 备份现有数据库（重要！）
```sql
-- 如果有重要数据，请先备份
mysqldump -u root -p online_exam > backup_online_exam.sql
```

### 2. 删除现有数据库
```sql
DROP DATABASE IF EXISTS online_exam;
```

### 3. 执行一键SQL文件
```bash
# 方法1：使用source命令
mysql -u root -p
source E:/1论文/在线考试系统 (AI功能毕业论文版)/在线考试系统/源代码/OnlineExamSystem/online_exam_one_click.sql

# 方法2：使用命令行导入
mysql -u root -p --default-character-set=utf8mb4 < "E:/1论文/在线考试系统 (AI功能毕业论文版)/在线考试系统/源代码/OnlineExamSystem/online_exam_one_click.sql"
```

### 4. 验证部署结果
执行以下查询验证数据：

## ⚠️ 注意事项

### 字符集兼容性
- 使用 `utf8` 字符集，与原系统保持一致
- 中文数据可能需要客户端字符集配置
- 英文测试数据确保功能正常运行

### 数据完整性
- 所有外键关系已正确设置
- 学号字段统一使用BIGINT类型
- 难度等级字段扩展为10字符

### 性能优化
- 添加了必要的索引
- 学生答题记录表包含复合索引
- 成绩表包含学生和考试类型索引

## 🔧 故障排除

### 导入失败
1. 检查MySQL版本兼容性
2. 确认字符集设置正确
3. 检查文件路径是否正确

### 中文显示问题
1. 设置MySQL客户端字符集：`SET NAMES utf8;`
2. 检查连接字符串配置
3. 使用英文测试数据验证功能

### 功能测试失败
1. 确认Spring Boot应用已重启
2. 检查数据库连接配置
3. 验证考试编号20230004存在
