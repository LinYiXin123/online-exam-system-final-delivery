# 在线考试系统 - 项目配置说明文档

## 📋 项目简介

这是一个基于 Spring Boot + MyBatis Plus 开发的在线考试系统后端项目。

- **项目名称**: OnlineExamSystemApi
- **框架版本**: Spring Boot 2.1.2.RELEASE
- **Java 版本**: JDK 1.8
- **数据库**: MySQL
- **端口**: 9201
- **构建工具**: Maven

## 📦 安装步骤

### 第一步：安装 JDK

1. 下载 JDK 1.8：https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html
2. 安装完成后，配置环境变量：
   - 新建系统变量 `JAVA_HOME`，值为 JDK 安装路径（如：`C:\Program Files\Java\jdk1.8.0_xxx`）
   - 在 `Path` 中添加 `%JAVA_HOME%\bin`
3. 验证安装：
```bash
java -version
javac -version
```

### 第二步：安装 Maven

1. 下载 Maven：https://maven.apache.org/download.cgi
2. 解压到指定目录（如：`D:\apache-maven-3.x.x`）
3. 配置环境变量：
   - 新建系统变量 `MAVEN_HOME`，值为 Maven 解压路径
   - 在 `Path` 中添加 `%MAVEN_HOME%\bin`
4. 验证安装：
```bash
mvn -version
```

### 第三步：安装 MySQL

1. 下载 MySQL：https://dev.mysql.com/downloads/mysql/
2. 安装并设置 root 密码（建议设置为 `123456`，与项目配置一致）
3. 启动 MySQL 服务：
```bash
# Windows 服务方式
net start mysql

# 或通过 MySQL Workbench 启动
```

## ⚙️ 项目配置

### 1. 修改数据库连接配置（如需要）

打开文件：`src/main/resources/application.properties`

```properties
# 服务器端口
server.port=9201

# 数据库用户名（根据你的 MySQL 配置修改）
spring.datasource.username=root

# 数据库密码（根据你的 MySQL 密码修改）
spring.datasource.password=123456

# 数据库连接地址
spring.datasource.url=jdbc:mysql://localhost:3306/online_exam?useUnicode=true&characterEncoding=UTF-8&useSSL=false&serverTimezone=UTC&allowPublicKeyRetrieval=true

# 数据库驱动
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver

# 数据源类型
spring.datasource.type=com.alibaba.druid.pool.DruidDataSource

# MyBatis 配置
mybatis.configuration.mapUnderscoreToCamelCase=true
logging.level.com.rabbiter.oes.mapper=info
mybatis-plus.configuration.log-impl=org.apache.ibatis.logging.stdout.StdOutImpl
```

## 🔍 常见问题

### 问题 1：端口被占用

**错误信息**：
```
Port 9201 is already in use
```

**解决方案**：
- 方案 A：修改 `application.properties` 中的 `server.port` 为其他端口（如 9202）
- 方案 B：关闭占用 9201 端口的程序

查找占用端口的程序：
```bash
# Windows
netstat -ano | findstr 9201

# 根据 PID 关闭进程
taskkill /F /PID <进程ID>
```

### 问题 2：数据库连接失败

**错误信息**：
```
Communications link failure
```

**解决方案**：
1. 确认 MySQL 服务已启动
2. 检查数据库用户名、密码是否正确
3. 确认数据库 `online_exam` 已创建
4. 检查防火墙是否阻止了 3306 端口

### 问题 3：Maven 依赖下载失败

**解决方案**：
1. 检查网络连接
2. 项目已配置阿里云镜像，下载速度应该较快
3. 如果还是很慢，可以手动配置 Maven 的 `settings.xml`

### 问题 4：找不到数据库

**错误信息**：
```
ERROR 1046 (3D000): No database selected
```

**解决方案**：
在导入 SQL 文件前，必须先执行：
```sql
CREATE DATABASE online_exam;
USE online_exam;
```
